Две прошивки для микроконтроллеров Raspberry Pico с использованием CircuitPython. Использовались в ВКР для иммитации устройств в CAN-сети.
Один микроконтроллер имитировал электромотор. А другой блок управления и индикатор поворота (на логическом уровне независимые устройства, разделение по фрагменту ID кадров).
